# Remote Support Home Lab

## Overview
Simulated a real-world help desk remote support environment using RustDesk
to remotely connect from a technician PC to a separate end-user PC on a local network.

## Tools Used
- RustDesk (remote desktop)
- Windows Task Manager
- Device Manager
- Event Viewer
- Windows %temp% file cleanup
- System Information

## What I Practiced
- Establishing and managing remote desktop sessions across two physical machines
- Monitoring and analyzing running processes for performance issues
- Auditing device drivers and flagging unknown hardware for follow-up
- Reviewing system event logs to identify and assess error patterns
- Performing temp file cleanup to maintain system performance
- Documenting hardware specs and system configuration
